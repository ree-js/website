# Welcome To Reejs Docs!

We are glad to have you here! This is the official documentation for Reejs. Here you will find everything you need to know about Reejs.

We expect you to know the basics of JavaScript and Node.js (or the runtime you choose!). If you don't know them, you can learn them [here](https://www.w3schools.com/js/default.asp) and [here](https://www.w3schools.com/nodejs/default.asp).

Wish you a happy learning!

P.S: You can contribute to this documentation [here](https://github.com/ree-js/website), we will be glad to have you!
Looking to contribute to the framework? Check out the [contributing guide](https://github.com/rovelstars/reejs/blob/main/CONTRIBUTING.md)!
